How the Internet Works

Basic Architecture

Client-Server Model

When you type www.example.com, your computer (client) requests data from a server, which responds with the webpage.

Protocols

  • HTTP/HTTPS for data transfer
  • DNS translates domain names to IP addresses

Browsing Efficiently

Browser Basics

Essential Shortcuts

  • Ctrl+T New Tab
  • Ctrl+W Close Tab
  • Ctrl+Shift+T Reopen Closed Tab

Advanced Search Techniques

Search Operators

  • Boolean: "web development AND best practices"
  • Exact Match: "digital literacy course"
  • Site Specific: site:edu "cybersecurity basics"

Security and Privacy

Privacy Settings

  • Use Ctrl+Shift+N for Incognito Mode
  • Regular cookie and cache clearing
  • Enable HTTPS-only mode
